# ERFrame ECG 🫀

**ERFrame ECG — Acil EKG Pratiği — İnteraktif Öğrenme Platformu**

Dr. Amir Reza Zirakpourazar

## Özellikler

- 35 EKG pattern (acilde göreceğin her şey)
- 280+ hasta varyasyonu (her pattern 8 farklı hasta)
- 12 derivasyon EKG görünümü
- 7 kategori (Temel, İskemi, Atriyal, Ventriküler, Blok, Metabolik, Diğer)
- **Öğren** modu: kategorize referans
- **Test Et** modu: rastgele quiz + skor takibi
- Klinik senaryolar + tanı kriterleri + tedavi protokolleri
- Mobil uyumlu
